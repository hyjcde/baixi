“融
通与超越 :清代诗学学术研讨会 ” 召开
2016年 11月 5 —6日，“融
通与超越:清代诗学学术研讨会 ” 在安徽师范大学召开 。本次会议由中国社会科
学院文学研究所古代文学研究室与教育部人文社会科学重点研究基地安徽师范大学中国诗学研究中心联合举办 。
来自中国社会科学院 、北京大学、南京大学、复旦大学、国家图书馆 、人民文学出版社等单位的学者就清代诗学
研究领域的问题进行深入讨论 。
一、以桐城派为代表的家族 、地域文学研究 。蒋寅 (华南师范大学 ) 论述方东树诗学的理论倾向和历史意
义。王达敏 (中国社会科学院 ) 分析新中国成立后桐城派文学精神与艺术追求的存续 。方盛良 (安徽大学 ) 从
“史” 的角度探讨 《文学研究法》
“文
学” 中深藏的 “经史” 之思。潘务正 (安徽师范大学 ) 对姚鼐与袁枚诗学
关系进行考察 。
二、清代诗学文献研究 。张寅彭 (上海大学) 展示了 《清诗话全编内编 》 顺治、康熙朝提要稿十篇 。周兴陆
(复旦大学) 从 《秋蟪吟馆诗钞》
《惜
阴书舍课艺 》 等文献中辑得晚清诗人金和诗文数篇 。孙微 (山东大学 ) 提
出道光间刊刻 《朱竹垞先生杜诗评本 》 乃伪托之作 。曾绍皇 (湖南师范大学 ) 探讨清代杜诗未刊评点与相关诗话
著作之间的联系 。陈丽丽 (河南大学) 梳理明清地域诗歌总集的编纂 ，并对其中的诗学思想进行概括 、分析。
三、清代作家作品个案及流派研究 。白一瑾 (北京大学 ) 论述龚鼎孳的文学观和诗歌特色 。刘一 (安徽大
学) 讨论冯班对齐梁体歌行的发展 。李朝军 (贵州师范大学 ) 对叶燮激进的仁政思想进行分析 。郭院林 (扬州大
学) 探讨乾隆西师与南巡诗的思想及其历史影响 。杨柏岭 (安徽师范大学 ) 阐释蒋春霖的儒家士子情怀及其词的
苦难书写。沙先一 (江苏师范大学 ) 讨论朱祖谋的词学取向 。周焕卿 (常州工学院 ) 探讨常州词派从师法姜夔 、
吴文英，到推崇苏轼的转变原因 。
四、清代诗学概念研究 。蔡德龙 (江苏第二师范学院 ) 讨论 “诗穷而后工 ” 说在清代的提升与越界 。郑升
(广西师范大学 ) 提出 “尚清” 的美学主张与袁枚 《随园诗话》 及清代中期诗学思想之关系 。唐芸芸 (重庆师范
大学) 对 “事境” 说进行阐释 。王宏林 (河南大学) 讨论 “诗言志”
“诗
缘情” 两个重要诗学观念在乾嘉时期的
离合。叶晔 (浙江大学) 通过对袁枚 “以棋喻诗” 说缘起、流变及 “易学难工” 的讨论，呈现清代文学思潮兴衰
更替的一个缩影 。罗时进 (苏州大学) 将 “差序格局” 概念引入文学史的观察视野 。
与会学者通过充分地交流 、积极地讨论 ，提出新观点 ，审视新材料 ，拓展了清代诗学研究视野 ，取得丰硕的
成果。
