# 第一节 "百戏"的概念界定与"汉唐"的时间界定

## 一、"百戏"的概念界定

### （一）百戏的基本概念

"百戏"是汉唐时期多种表演伎艺之总称【1】，又称"角觝戏""散乐"【2】，是中古时期极为流行的艺术表演形式，其中包括了各种各样的技艺【3】。从词源上看，"百"言其技艺种类之繁多，"戏"则强调其表演娱乐之属性。

百戏的内涵代有不同，管理的机构更随王朝的更迭、时世的迁延而发生变化【4】。就管理机构而言，两汉有乐府、黄门鼓吹、百戏师；魏晋南朝有黄门鼓吹、太乐署；北朝有太乐署、鼓吹署、总章署；唐代则主要由教坊管理【5】。

### （二）百戏的构成与性质

关于百戏的构成，学界存在一定争议。任半塘先生在其影响广泛的《唐戏弄》一书中，对"戏弄、百戏、戏剧"三者作出如下定义：

> "所谓戏弄，显有两部分：一为百戏，表现体力、技巧，限于感官，以动目眩神，令人诧异为止……一为戏剧，表现人生，模拟精神，出入于言语、行动、表情之中。"【6】

然而，黎国韬通过对两汉"鱼龙幻化"等具体艺术形式的考证，对此定义提出了修正意见。他指出，任先生的定义"与古文献记载多不符"【7】，因为如《通典·乐六》所载，北齐、北周散乐百戏中便有"俳优、朱儒、山车、巨象、拔井、种瓜、杀马、剥驴等"【8】，其中"俳优"明显属戏剧范畴，"拔井、种瓜"等亦含幻术性质的戏剧表演。

据此，黎国韬提出修正的定义：

> "百戏实由戏剧与杂戏（或戏弄）两大子项构成，百戏与戏剧不应为'对立'关系，而应为'包含'关系。"【9】

具体而言：

- **"百戏"**可视为汉唐时期多种表演伎艺之总称，大致相当《唐戏弄》所谓之"戏弄"；
- **"戏剧"**则为百戏中一子项，定义大致相当今天所谓之"话剧"或"戏剧"；
- **"杂戏"**（或"戏弄"）指戏剧以外之百戏表演，内涵相当于《唐戏弄》所谓之"百戏"【10】。

## 二、"汉唐百戏"的时间界定

### （一）为何以汉代为研究起点

"百戏"虽然是汉唐时期多种表演伎艺的总称，但这并不意味着汉代以前就没有类似的表演活动。事实上，"散乐"一词早见于《周礼》，郑玄注云："散乐，野人为乐之善者，若今黄门倡矣，自有舞。"【80】然而，先秦时期的表演活动与汉代百戏存在本质差异，主要体现在以下几个方面：

**第一，综合性特征尚未形成。**

汉代百戏最显著的特征是"多伎杂陈"的综合性，即在同一场合中演出多种不同类型的伎艺。这种综合性在先秦时期尚未出现。《隋书·音乐志》所载"始（北）齐武平中，有鱼龙烂漫、俳优、朱儒、山车、巨象、拔井、种瓜、杀马、剥驴等，奇怪异端，百有余物，名为'百戏'"【81】，"百有余物"正是强调这种综合性特征。

**第二，西域幻术尚未传入。**

据杜佑《通典·乐六》记载："大抵散乐杂戏多幻术，皆出西域，始于善幻人至中国。"【82】西域幻术的大规模传入是在西汉武帝张骞凿空西域之后。《史记·大宛列传》记载："及加其眩者之工，而觳抵奇戏岁增变，甚盛益兴，自此始。"【83】可见，百戏的兴盛与西域文化交流密切相关，这是先秦时期所不具备的条件。

**第三，专门管理机构尚未设立。**

两汉时期设立了专门管理百戏的机构，如乐府、黄门鼓吹、百戏师等【84】。这种制度化的管理模式标志着百戏已成为一种成熟的表演艺术形态。先秦时期的"散乐"则主要依附于宗教仪式或礼乐活动，缺乏独立的管理体系。

因此，虽然先秦时期已有"散乐"等表演活动，但这些活动尚未发展成为具有综合性、世俗娱乐性、制度化特征的"百戏"。以汉代作为百戏研究的起点，既符合百戏发展的历史实际，也有利于界定研究对象的内涵。

### （二）汉唐百戏的整体性

将"汉唐"作为百戏研究的时间范畴，是基于这一时期百戏发展的整体性和连续性：

**第一，管理机构的延续性。**

从两汉至唐代，百戏的管理机构虽有更迭，但始终保持着制度上的连续性。两汉有乐府、黄门鼓吹、百戏师；魏晋南朝有黄门鼓吹、太乐署；北朝有太乐署、鼓吹署、总章署；唐代则主要由教坊管理【85】。这种管理机构的延续性，表明百戏在这一时期始终作为一种官方认可的表演艺术形态而存在。

**第二，表演内容的一致性。**

《魏书·乐志》载北魏道武帝天兴六年（403）冬诏书云："太乐、总章、鼓吹增修杂伎，造五兵、角觝、麒麟、凤皇、仙人、长蛇、白象、白虎及诸畏兽、鱼龙、辟邪、鹿马仙车、高絙百尺、长趫、缘橦、跳丸、五案以备百戏。大飨设之于殿庭，**如汉晋之旧也**。"【86】"如汉晋之旧也"一语，明确说明北魏时期的百戏沿袭了汉晋传统，表演内容并无本质差异。

类似地，《三国志》裴注引《魏略》载魏明帝青龙三年"使博士马钧作司南车，水转百戏。岁首建巨兽、鱼龙、曼延，弄马倒骑，**备如汉西京之制**"【87】。"备如汉西京之制"同样表明百戏传承的连续性。

**第三，"散乐"概念的稳定性。**

在汉唐时期，"散乐"与"百戏"基本同义，均指"多伎杂陈"的综合性表演。黎国韬指出："角觝戏即'百戏'，又称'散乐'，是中古时期多种表演伎艺的总称。"【88】这种概念上的稳定性，表明汉唐时期的百戏在性质上具有一致性。

### （三）为何以唐代为研究下限

以唐代作为百戏研究的下限，主要基于宋代以后百戏发生的根本性变化：

**第一，各伎艺的分化与专业化。**

宋代以后，原本统一于"百戏"名下的各种伎艺开始分化、独立，走向专业化发展的道路。例如，杂技、幻术、傀儡戏、说唱等原属百戏的伎艺，在宋代逐渐发展成为独立的表演门类，各有专门的艺人群体和演出场所。吴自牧《梦粱录》"百戏伎艺"条虽仍记载"水百戏""鱼龙变化"等表演【89】，但这些伎艺已不再是综合性的百戏演出，而是各自独立的专业表演。

**第二，戏剧的独立与成熟。**

在汉唐时期，戏剧是百戏的一个子项，如黎国韬所论"百戏实由戏剧与杂戏（或戏弄）两大子项构成"【90】。然而，宋元以后，杂剧、南戏等戏剧形式迅速发展成熟，从百戏中独立出来，成为一种具有独立审美价值和表演体系的艺术形态。这种变化标志着百戏作为综合性表演形式的历史使命基本完成。

**第三，"散乐"概念的演变。**

隋唐时期，"散乐"作为与"百戏"同义的术语，指称综合性的伎艺表演。《隋书·音乐志》云："周时，郑译有宠于宣帝，奏征齐散乐人，并会京师为之。盖秦角抵之流者也。"【91】可见，隋代的"散乐"仍指综合性百戏。但宋代以后，"散乐"的内涵发生变化，逐渐专指某些特定的伎艺类型，不再与"百戏"同义。

**第四，表演场所的变化。**

汉唐时期，百戏主要在宫廷和官方场合演出，由专门的乐官机构管理。宋代以后，随着瓦舍勾栏等商业性演出场所的兴起，各类伎艺开始面向市民阶层进行商业演出，原本统一的官方管理体系逐渐解体，各伎艺走向独立发展。

综上所述，宋代以后百戏经历了从"综合化"向"专业化"的根本转型，"百戏"作为综合性表演形式的概念逐渐淡化，各伎艺走向独立发展的道路。因此，以唐代作为百戏研究的下限，既符合百戏发展的历史规律，也有利于把握百戏的本质特征。

### （四）百戏发展的阶段划分

综合上述分析，百戏的发展可划分为以下几个阶段：

| 阶段       | 时间范围       | 特征                                 |
| ---------- | -------------- | ------------------------------------ |
| **萌芽期** | 先秦           | "散乐"见于典籍，但尚未形成综合性特征 |
| **兴起期** | 西汉武帝至东汉 | 西域幻术传入，百戏兴盛，管理机构设立 |
| **发展期** | 魏晋南北朝     | "如汉晋之旧"，百戏传承延续           |
| **鼎盛期** | 隋唐           | 教坊管理，百戏达到高峰               |
| **分化期** | 宋代以后       | 各伎艺专业化，百戏概念逐渐淡化       |

其中，**汉代至唐代**（包括魏晋南北朝）构成了百戏发展的核心时期，这一时期的百戏具有以下共同特征：

1. **综合性**：多种伎艺在同一场合演出
2. **制度化**：有专门的官方管理机构
3. **概念统一性**："百戏""散乐""角觝戏"基本同义
4. **传承连续性**：各朝代间存在明确的继承关系

因此，将"汉唐"作为百戏研究的时间范畴，是基于百戏发展规律的科学界定。

## 三、百戏的具体类型

根据文献记载和考古材料，汉唐百戏主要包括以下几种类型：

### （一）幻术类

幻术是百戏中最具代表性的类型之一。据杜佑《通典·乐六》记载："大抵散乐杂戏多幻术，皆出西域，始于善幻人至中国。"【11】最著名的幻术表演当属"鱼龙幻化"，据《后汉书·礼仪志》引《汉官典职》记载：

> "作九宾乐。舍利之兽从西方来，戏于庭，入前殿，激水化成比目鱼，嗽水作雾，化成黄龙，长八丈，出水遨戏于庭，炫濯日光。"【12】

此外，《隋书·音乐志》亦载北齐、北周百戏中有"拔井、种瓜、杀马、剥驴"等幻术表演【13】。

### （二）假面（头）兽舞

假面兽舞由"象人"表演，是百戏的重要组成部分。据《汉书·礼乐志》记载，西汉乐府中设有"常从象人四人""秦倡象人员三人"【14】。孟康注《汉书》曰："象人，若今戏虾、鱼、师子者也。"韦昭注则云："著假面者也。"【15】由此可知，象人是戴着假面（头）而作百戏兽舞之人。

张衡《西京赋》对此有生动描绘："总会仙倡，戏豹舞羆。白虎鼓瑟，苍龙吹箎。"李善注曰："仙倡，伪作假形，谓如神也。羆豹熊虎，皆为假头也。"【16】

东汉以后，象人逐渐演变为"百戏师"。据《唐六典》记载："后汉少府，属官有承华令：典黄门鼓吹，百三十五人；百戏师，二十七人。"【17】黎国韬指出："从名称上可以推断，百戏师应当是承担百戏表演的乐师"，"所以表演百戏的百戏师应当是从象人转变而来的"【18】。

### （三）傀儡戏

傀儡戏可分为假人傀儡戏和假面（头）傀儡戏两大类型【19】。

**假人傀儡戏**指真人利用机关或其它各种手段（诸如牵丝、杖头、指头等）操控假人（包括各种材质的假人，诸如木偶、铜偶、土偶等）而作表演的伎艺，大致相当于今天所说的木偶戏【20】。如《西京杂记》记载秦宫遗宝：

> "铸铜人十二枚，坐皆高三尺，列在一筵上，琴筑笙竽各有所执。皆缀花采，俨若生人。筵下有二铜管，上口高数尺，出筵后。其一管空，一管内有绳大如指。使一人吹空管，一人纽绳，则众乐皆作，与真乐不异焉。"【21】

三国时马钧所作"水转百戏"，亦属此类【22】。

**假面（头）傀儡戏**则指真人戴上假面（头）而作舞蹈、戏剧、戏弄等表演的伎艺，大致相当于今天所说的面具表演艺术【23】。

### （四）杂技类

杂技类百戏种类繁多，张衡《西京赋》有详细记载：

> "临迴望之广场，程角觝之妙戏：乌获扛鼎，都卢寻橦……尔乃建戏车，树修旃。侲僮程材，上下翩翻。"【24】

具体包括：都卢寻橦（竿技）、走索（绳伎）、缘橦、跳丸、弄剑、透剑、顶竿、倒立等【25】。山东沂南东汉画像石中亦刻画有透剑、顶竿、盘舞、绳伎、倒立、缘橦等杂技表演场面【26】。

### （五）角力类

角抵（相扑）是百戏中具有竞技性质的表演形式。据《汉书·武帝纪》记载："（元封）三年（前108年）春，作角抵戏，三百里内皆［来］观。"【27】又载："（元封）六年（前105年）……夏，京师民观角抵于上林平乐馆。"【28】

### （六）舞蹈类

百戏中的舞蹈类表演与宫廷傩仪密切相关。据《后汉书·礼仪志》记载，东汉大傩仪中有侲子舞、方相舞、十二兽舞等【29】。张衡《西京赋》所述"侲僮程材，上下翩翻"【30】，薛综注云："侲子言善也，善童幼子也。"【31】可见侲僮既善于"角觝戏"，即后世常称的"百戏"，其身份属倡优一类【32】。

### （七）动物戏弄

动物戏弄包括戏豹、戏凤、戏马、马戏、斗虎等。《盐铁论·散不足》记载："今民间……戏弄蒲人杂妇，百兽马戏斗虎。"【33】山东沂南东汉画像石中亦有戏豹、戏凤、戏马等表演场面的刻画【34】。

## 四、百戏与相关术语的关系辨析

### （一）百戏与散乐

"散乐"一词见于《周礼》。据《周礼·旄人》郑玄注："散乐，野人为乐之善者，若今黄门倡矣，自有舞。"【35】由此可知，散乐原指非雅乐的民间娱乐表演，与宫廷黄门倡优的表演性质相近。

从历史演变看，"百戏"与"散乐"基本同义。黎国韬指出："角觝戏即'百戏'，又称'散乐'，是中古时期多种表演伎艺的总称。"【36】但二者侧重点略有不同：

| 术语     | 侧重点                 | 时代用法     |
| -------- | ---------------------- | ------------ |
| **百戏** | 强调技艺种类之繁多     | 汉唐普遍使用 |
| **散乐** | 强调"野人为乐"的民间性 | 隋唐后更常用 |

此外，杜佑《通典·乐六》指出散乐与西域的密切关系："大抵散乐杂戏多幻术，皆出西域，始于善幻人至中国。汉安帝时，天竺献伎，能自断手足，刳剔肠胃，自是历代有之。"【37】

### （二）百戏与角抵（角觝）

"角抵"一词有狭义和广义之分。狭义的角抵专指相扑格斗；广义的角抵则与"百戏"同义。

据《汉书·西域传》记载，汉武帝"作《巴俞》都卢、海中《砀极》、漫衍、鱼龙、角抵之戏以观视之"【38】。颜师古注"角抵"云："角者，角技也；抵者，相抵触也。"【39】此为狭义。

然而，张衡《西京赋》云"程角觝之妙戏"，却包含了都卢寻橦、走索、弄剑、跳丸、鱼龙幻化等多种表演【40】。黎国韬指出："广义角觝戏则相当于'百戏'，又称'散乐'。"【41】

三者关系可概括如下：

| 术语     | 狭义     | 广义             |
| -------- | -------- | ---------------- |
| **角抵** | 相扑格斗 | 与百戏同义       |
| **百戏** | —        | 多种表演伎艺总称 |
| **散乐** | —        | 与百戏同义       |

### （三）百戏与傩仪、傩戏

#### ① 傩仪和傩戏的概念界定

**傩仪**是中国古代一种驱疫逐鬼的宗教祭祀仪式。据《周礼·夏官》记载，周代大傩仪的核心角色为方相氏：

> "方相氏，狂夫四人。方相氏，掌蒙熊皮，黄金四目，玄衣朱裳，执戈扬盾，帅百隶而时难，以索室殴疫。"【42】

周代大傩仪中，方相氏的"狂夫"之舞带有迷狂状态的萨满性质，巫术色彩非常浓厚【43】。其执事官主要有方相氏、百隶，还有占梦、男巫、女祝等，"多数是巫职人员"【44】。

**傩戏**则是从傩仪转变而成的戏剧形态。黎国韬指出："古代傩仪和傩戏是戏剧史、民俗史研究的热点话题之一，学者普遍认为傩戏是从傩仪转变而成的。"【45】傩仪向傩戏的转变，"关系到宗教仪式向戏剧形态转变和发展这一重大的理论课题"【46】。

东汉禁中大傩仪的记载见于《后汉书·礼仪志》：

> "先腊一日，大傩，谓之逐疫。其仪，选中黄门子弟年十岁以上，十二以下，百二十人为侲子。皆赤帻皀制，执大鼗。方相氏黄金四目，蒙熊皮，玄衣朱裳，执戈扬盾。十二兽有衣毛角。中黄门行之，冗从仆射将之，以逐恶鬼于禁中……因作方相与十二兽舞。嚾呼，周遍前后省三过，持炬火，送疫出端门。"【47】

相较于周代大傩仪以"狂夫"之舞为核心，东汉大傩仪已转变为"方相、十二兽、侲子之嚾呼、群舞作为仪式的核心表演，娱乐成分较重"【48】。

#### ② 百戏与傩之间的共同点

百戏与傩仪、傩戏之间存在密切的交叉关系，主要体现在以下几个方面：

**第一，表演形式的相似性。**

百戏与傩仪都包含假面（头）表演这一重要形式。百戏中的"象人"是"戴着假面（头）而作百戏兽舞之人"【49】，孟康注《汉书》曰："象人，若今戏虾、鱼、师子者也。"韦昭注则云："著假面者也。"【50】而傩仪中的方相氏同样"蒙熊皮，黄金四目"【51】，十二兽亦"有衣毛角"【52】，均属假面表演性质。

孙楷第早已指出傀儡戏（含假面表演）与傩的关系："近代傀儡有二派，一派以真人扮演，如宋之傀儡'舞鲍老'、'耍和尚'等是也。'舞鲍老'、'耍和尚'戴假首，与汉之舞方相同。"【53】

**第二，执事人员的重叠。**

东汉以后，傩仪的执事者逐渐与百戏表演者合流。黎国韬考证指出：

> "扮演方相和十二兽的当是象人和百戏师，他们也是两汉宫廷乐官之属，而且与黄门乐官同样有密切关系。"【54】

具体而言，东汉禁中大傩仪的主要执事官包括中黄门子弟、中黄门、中黄门冗从仆射、黄门令、象人、百戏师等【55】。其中，扮演侲子的中黄门子弟善于"角觝戏"，"也就是后世常称的'百戏'，其身份属倡优一类"【56】。张衡《西京赋》云："临迴望之广场，程角觝之妙戏……侲僮程材，上下翩翻。"【57】薛综注云："侲子言善也，善童幼子也。"【58】

**第三，管理机构的统一。**

据《唐六典》《通典》记载，后汉设有承华令，"典黄门鼓吹，百三十五人；百戏师，二十七人"【59】。黄门鼓吹既是百戏的主要管理和表演机构，同时也是东汉大傩仪的主要执事机构。蔡邕《乐意》称："三曰黄门鼓吹，天子所以宴乐群臣。"【60】

**第四，娱乐功能的共通。**

百戏与傩仪在历史发展中都呈现出娱乐化的趋势。黎国韬指出："由于有黄门乐人、黄门倡优的主持和参与，使东汉禁中大傩仪之世俗性和娱乐成分较前代大为增强。"【61】

#### ③ 百戏与傩之间的不同点

尽管百戏与傩仪存在诸多交叉，但二者在本质属性、功能目的、历史演变等方面仍有根本区别：

**第一，本质属性不同。**

| 比较维度     | 百戏               | 傩仪/傩戏          |
| ------------ | ------------------ | ------------------ |
| **性质**     | 世俗娱乐性表演     | 宗教仪式性表演     |
| **核心特征** | 技艺展示、娱人耳目 | 驱疫逐鬼、宗教祭祀 |
| **巫术色彩** | 较弱或无           | 周代浓厚，东汉渐弱 |

百戏以世俗娱乐为主要功能，强调技艺的展示和观赏的愉悦。傩仪则以宗教祭祀、驱疫逐鬼为核心目的。《后汉书·礼仪志》明确记载傩仪的功能是"逐疫""逐恶鬼于禁中"【62】。

**第二，执事者身份的历史变迁。**

周代大傩仪的执事官"多数是巫职人员"【63】，包括方相氏、百隶、占梦、男巫、女祝等。然而，"这些巫职人员在东汉禁中大傩仪的实施过程中却一个也找不到"【64】。东汉大傩仪的执事者已完全转变为黄门乐人、倡优。

黎国韬对此做出深刻分析：

> "以巫师为中心的大傩仪已经完全转变成为以乐官为中心，可以说是一种质的转变；而这也正是限制傩仪宗教色彩、增强傩仪娱乐成分的最直接的、更深层的原因。"【65】

**第三，发展路径不同。**

百戏的发展呈现出**"综合化"**趋势，不断吸收西域幻术、杂技、舞蹈等多种表演形式。据《史记·大宛列传》记载，西汉武帝时"及加其眩者之工，而觳抵奇戏岁增变，甚盛益兴"【66】。

傩仪的发展则呈现出**"戏剧化"**趋势，逐渐从宗教仪式向戏剧表演转化。黎国韬指出：

> "傩仪为什么会向傩戏转变？这种转变最早在何时发生？又是如何发生的？……个人认为，这种转变必须以仪式中娱乐成分的大量增强为前提，而对东汉禁中大傩仪主要执事官身份的辨析，恰好揭示了这种仪式当中带有较强娱乐成分的直接原因。"【67】

**第四，戏剧史意义不同。**

百戏作为汉唐时期多种表演伎艺的总称，其中"至少存在两种戏剧"【68】，如"鱼龙幻化"等，与古代戏剧的形成和发展存在直接联系【69】。

傩仪向傩戏的转变则代表了"宗教仪式向戏剧形态转变和发展"【70】这一独特路径。黎国韬指出：

> "虽然从现有史料来看，古代傩仪之转变为古代戏剧不是一朝一夕完成的，但迈出转变的第一步至迟从东汉禁中大傩仪实施时就已经开始了。"【71】

**第五，流传范围不同。**

百戏主要在宫廷和都市中流行，由专门的乐官机构管理。傩仪、傩戏则既有宫廷形态，也广泛流传于民间，如《后汉书·礼仪志》所载"百官官府各木面兽能为傩人师"【72】，说明各级官府乃至民间都有傩仪的施行。

综上所述，百戏与傩仪虽在表演形式、执事人员、管理机构等方面存在交叉融合，但在本质属性、功能目的、发展路径上仍有根本区别。东汉以后，由于黄门乐人、百戏师等倡优群体的介入，傩仪的宗教色彩逐渐削弱，娱乐成分不断增强，从而为傩仪向傩戏的转变奠定了基础。这一历史进程，深刻揭示了中国早期戏剧发展的复杂性与多元性。

## 五、百戏的历史演变

### （一）西汉：百戏的兴起

西汉武帝时期是百戏发展的关键时期。《史记·大宛列传》记载：

> "是时上（汉武帝）方数巡狩海上，乃悉从外国客，大都多人则过之，散财帛以赏赐，厚具以饶给之，以览示汉富厚焉。于是大觳抵，出奇戏诸怪物，多聚观者，行赏赐，酒池肉林，令外国宾客遍观各仓库府藏之积，见汉之广大，倾骇之。及加其眩者之工，而觳抵奇戏岁增变，甚盛益兴，自此始。"【73】

可见，西汉角觝戏（百戏）的兴盛与张骞凿空西域后中外文化交流的频繁密切相关，特别是西域"黎轩善眩人"来华后，"觳抵奇戏岁增变，甚盛益兴"【74】。

### （二）东汉：百戏的制度化

东汉时期，百戏管理趋于制度化。据《唐六典》《通典》记载，后汉设有承华令，"典黄门鼓吹，百三十五人；百戏师，二十七人"【75】。

《后汉书·安帝纪》载："（延平元年十二月）乙酉，罢鱼龙曼延百戏。"【76】李贤注引《汉官典职》详述了百戏的具体内容，表明"鱼龙""曼延""舍利"等表演是后汉百戏的主要项目。黎国韬据此指出：

> "后汉所谓的'百戏'主要是指舞弄'鱼龙'、'曼延'、'舍利'等的表演，这与前文提到的象人戴假面（头）以戏鱼、虾、狮子基本上没有区别。"【77】

### （三）魏晋南北朝：百戏的延续

魏晋南北朝时期，百戏继续传承发展。《三国志》裴注引《魏略》载：

> "是年（明帝青龙三年）起太极诸殿，筑总章观。……使博士马钧作司南车，水转百戏。岁首建巨兽、鱼龙、曼延，弄马倒骑，备如汉西京之制。"【78】

《魏书·乐志》载北魏道武帝天兴六年（403）冬诏书云：

> "太乐、总章、鼓吹增修杂伎，造五兵、角觝、麒麟、凤皇、仙人、长蛇、白象、白虎及诸畏兽、鱼龙、辟邪、鹿马仙车、高絙百尺、长趫、缘橦、跳丸、五案以备百戏。大飨设之于殿庭，如汉晋之旧也。"【79】

## 六、小结

综上所述，"百戏"是汉唐时期多种表演伎艺的总称，与"散乐""角抵"（广义）基本同义，包含幻术、假面兽舞、傀儡戏、杂技、角力、舞蹈、动物戏弄等多种类型。在概念构成上，百戏实由戏剧与杂戏（或戏弄）两大子项构成，百戏与戏剧是"包含"而非"对立"关系。

百戏的发展具有以下特点：第一，综合性，包含多种表演伎艺；第二，世俗娱乐性，主要功能是娱乐；第三，国际性，大量吸收西域幻术和表演艺术；第四，历史延续性，从西汉至明清持续传承。百戏与傩仪的交叉融合，使东汉大傩仪的执事者从巫职人员转变为黄门乐人，这是傩仪向傩戏转化的重要原因之一。

# 第二节 "汉唐百戏"的演出类型

## 一、汉唐百戏演出类型概述

### （一）学界的主要分类方式

关于汉唐百戏的演出类型，学界尚无统一的分类标准。综观既有研究，主要存在以下几种分类方式：

**第一，按表演性质分类。**

任半塘在《唐戏弄》中将"戏弄"（即广义的百戏）分为两大类：一为"百戏"，表现体力、技巧；一为"戏剧"，表现人生、模拟精神【92】。萧亢达《汉代乐舞百戏艺术研究》则将百戏分为杂技、幻术、角抵、俳优、象人之戏、动物表演等类型【93】。

**第二，按文献记载分类。**

据《隋书·音乐志》所载，北齐、北周散乐百戏包括"鱼龙烂漫、俳优、朱儒、山车、巨象、拔井、种瓜、杀马、剥驴等，奇怪异端，百有余物"【94】。《魏书·乐志》则列举了"五兵、角觝、麒麟、凤皇、仙人、长蛇、白象、白虎及诸畏兽、鱼龙、辟邪、鹿马仙车、高絙百尺、长趫、缘橦、跳丸、五案"等项目【95】。

**第三，按表演对象分类。**

从表演主体的角度，汉唐百戏可分为：以人为主体的表演（如杂技、幻术）、以动物为主体的表演（如马戏、斗虎）、人与动物结合的表演（如假面兽舞）。

### （二）本文的分类标准与选取依据

鉴于汉唐百戏种类繁多、性质各异，本文不拟对所有类型进行穷尽式研究，而是选取**幻术、杂技、拟兽、驯兽**四种类型作为考察对象。选取依据如下：

**第一，代表性。**

这四种类型涵盖了汉唐百戏的主要表演形态，既有人的技艺展示（杂技、幻术），也有人与动物的互动（拟兽、驯兽），具有较强的代表性。

**第二，文献丰富度。**

这四种类型在汉唐文献中留下了大量记载，包括史书、政书、文学作品等多种形式的"百戏书写"，便于进行系统的文本分析。

**第三，研究价值。**

这四种类型各具特色，能够从不同角度揭示汉唐百戏的艺术特征、文化内涵和社会功能。

## 二、本文选取的四种演出类型

### （一）幻术

**幻术**是汉唐百戏中最具代表性的类型之一，以变幻莫测的视觉效果为核心特征。据杜佑《通典·乐六》记载："大抵散乐杂戏多幻术，皆出西域，始于善幻人至中国。汉安帝时，天竺献伎，能自断手足，刳剔肠胃，自是历代有之。"【96】

幻术的主要内容包括：

| 类型       | 代表节目               | 文献出处                         |
| ---------- | ---------------------- | -------------------------------- |
| **变化类** | 鱼龙幻化、黄龙变       | 《后汉书·礼仪志》《隋书·音乐志》 |
| **幻象类** | 吞刀、吐火、自支解     | 《通典·乐六》                    |
| **机关类** | 拔井、种瓜、杀马、剥驴 | 《隋书·音乐志》                  |

其中，"**鱼龙幻化**"是汉唐幻术的代表性节目。据《后汉书·礼仪志》引《汉官典职》记载："作九宾乐。舍利之兽从西方来，戏于庭，入前殿，激水化成比目鱼，嗽水作雾，化成黄龙，长八丈，出水遨戏于庭，炫濯日光。"【97】这种以"鱼化为龙"为主题的幻术表演，融合了西域幻术技法与中国龙文化意象，具有重要的戏剧史意义【98】。

### （二）杂技

**杂技**是汉唐百戏中最基础、最普遍的类型，以展示人体技巧、力量和协调性为核心。张衡《西京赋》对汉代杂技有详尽描绘：

> "临迴望之广场，程角觝之妙戏：乌获扛鼎，都卢寻橦……尔乃建戏车，树修旃。侲僮程材，上下翩翻。突倒投而跟挂，譬陨绝而复联。"【99】

汉唐杂技的主要类型包括：

| 类型     | 具体内容             | 特征                             |
| -------- | -------------------- | -------------------------------- |
| **竿技** | 都卢寻橦、缘橦、顶竿 | 以高竿为道具，展示攀爬、平衡技巧 |
| **绳技** | 走索、绳伎           | 在绳索上行走、表演               |
| **力技** | 扛鼎、角力           | 展示力量                         |
| **手技** | 跳丸、弄剑、透剑     | 展示手部灵巧                     |
| **身技** | 倒立、翻筋斗         | 展示身体柔韧与协调               |

山东沂南东汉画像石中刻画有透剑、顶竿、盘舞、绳伎、倒立、缘橦等杂技表演场面【100】，为我们提供了汉代杂技表演的直观图像资料。

### （三）拟兽

**拟兽**是指由人扮演动物形象进行表演的伎艺，又称"假面兽舞""象人之戏"。其核心特征是：表演者戴上动物造型的假面（头），穿上模拟动物的服装，扮作各种禽兽进行舞蹈或戏剧表演。

据《汉书·礼乐志》记载，西汉乐府中设有"常从象人四人""秦倡象人员三人"【101】。孟康注曰："象人，若今戏虾、鱼、师子者也。"韦昭注云："著假面者也。"【102】可见，"象人"是专门从事假面兽舞的百戏艺人。

张衡《西京赋》描绘了拟兽表演的盛况："总会仙倡，戏豹舞羆。白虎鼓瑟，苍龙吹箎。"李善注曰："仙倡，伪作假形，谓如神也。羆豹熊虎，皆为假头也。"【103】

拟兽表演的主要类型：

| 扮演对象         | 表演形式       | 文化意涵       |
| ---------------- | -------------- | -------------- |
| **龙**           | 鱼龙幻化、龙舞 | 祥瑞、帝王象征 |
| **狮子**         | 狮子舞         | 西域传入，辟邪 |
| **熊、虎、豹**   | 假面兽舞       | 力量、威猛     |
| **凤凰、麒麟**   | 瑞兽舞         | 祥瑞、太平     |
| **方相、十二兽** | 傩仪表演       | 驱疫逐鬼       |

拟兽表演与傩仪存在密切联系。据《后汉书·礼仪志》记载，东汉大傩仪中"方相氏黄金四目，蒙熊皮，玄衣朱裳""十二兽有衣毛角"【104】，其表演形式实属拟兽范畴。

### （四）驯兽

**驯兽**是指对真实动物进行训练，使其表演各种技艺的伎艺，又称"戏弄""驯养""动物戏弄"。与"拟兽"由人扮演动物不同，"驯兽"的表演主体是经过训练的真实动物。

《盐铁论·散不足》记载："今民间……戏弄蒲人杂妇，百兽马戏斗虎。"【105】所谓"百兽马戏斗虎"，即是驯兽表演的典型内容。

汉唐驯兽表演的主要类型：

| 动物种类       | 表演形式             | 文献记载                 |
| -------------- | -------------------- | ------------------------ |
| **马**         | 马戏、弄马倒骑、戏马 | 《三国志》《盐铁论》     |
| **象**         | 山车巨象             | 《隋书·音乐志》          |
| **虎、豹**     | 斗虎、戏豹           | 《盐铁论》《沂南画像石》 |
| **凤（禽类）** | 戏凤                 | 《沂南画像石》           |
| **鱼**         | 弄龟伎               | 《隋书·音乐志》          |

山东沂南东汉画像石中刻画有戏豹、戏凤、戏马等表演场面【106】，表明汉代驯兽表演已相当发达。

### （五）四种类型的关系

**幻术、杂技、拟兽、驯兽**四种类型并非彼此孤立，而是常常相互融合、共同演出。例如：

- **"鱼龙幻化"**融合了幻术（变化效果）与拟兽（人扮鱼、龙）；
- **竿技**中的"都卢寻橦"常与拟兽表演结合；
- **傩仪**中的十二兽舞既属拟兽，又包含舞蹈（杂技范畴）元素。

这四种类型的综合运用，正是汉唐百戏"多伎杂陈"综合性特征的具体体现。

# 第三节 "百戏书写"的概念及类型

## 一、"书写"的概念界定

### （一）学术语境下的"书写"

"书写"（writing）作为学术概念，在不同学科语境下具有不同的内涵。在文学研究领域，"书写"通常指对特定对象、主题或经验的文本表达与建构。这一概念强调的不仅是记录行为本身，更包括记录者的立场、视角、策略以及文本所承载的意义。

当代学术研究中，"书写"已成为一种重要的分析范畴。例如，"历史书写"关注历史叙述的建构过程及其背后的权力关系；"身体书写"关注文学作品中对身体的表达与建构；"空间书写"则聚焦于文本中空间意象的呈现与意义。

### （二）"书写"与相关概念的辨析

"书写"与"记载""描写""叙事"等概念既有联系，又有区别：

| 概念     | 侧重点   | 特征                     |
| -------- | -------- | ------------------------ |
| **记载** | 客观记录 | 强调信息的保存与传递     |
| **描写** | 艺术再现 | 强调形象的生动性         |
| **叙事** | 故事讲述 | 强调情节的组织与展开     |
| **书写** | 文本建构 | 强调意义的生产与表达策略 |

相较于"记载""描写""叙事"等概念，"书写"更具批判性和反思性，它提醒我们关注文本背后的作者意图、社会语境和意识形态因素。

## 二、"百戏书写"的界定

### （一）什么是"百戏书写"

**"百戏书写"**是指古代文献中对百戏及其相关内容的文本表达与建构，包括对百戏演出场景的描绘、对百戏伎艺的记录、对百戏艺人的叙述、对百戏制度的载录，以及由百戏引发的议论与评价等。

具体而言，"百戏书写"涵盖以下内容：

1. **演出场景**：对百戏表演的具体情境、观演关系的描绘；
2. **伎艺内容**：对各类百戏伎艺的名称、特征、技法的记录；
3. **艺人群体**：对百戏表演者的身份、地位、生活的叙述；
4. **管理制度**：对百戏管理机构、演出规制的载录；
5. **社会评价**：对百戏的功过是非、文化意义的议论。

### （二）"百戏书写"的研究价值

研究"百戏书写"具有以下学术意义：

**第一，还原历史实态。**

百戏作为一种表演艺术，其演出过程转瞬即逝，无法直接保存。我们对汉唐百戏的认识，主要依赖于古代文献的"书写"。通过对"百戏书写"的系统考察，可以尽可能还原汉唐百戏的历史面貌。

**第二，揭示文化意涵。**

不同类型的文献对百戏的书写方式存在差异，这些差异反映了不同书写主体的立场、视角和价值取向。通过比较分析，可以揭示百戏在汉唐社会中的文化定位和意义建构。

**第三，拓展研究视野。**

"百戏书写"这一概念的引入，有助于突破传统百戏研究主要聚焦于"百戏本身"的局限，将研究视野扩展到"关于百戏的书写"层面，从而深化对汉唐百戏的理解。

## 三、百戏书写的类型

根据书写主体、文本性质和功能目的的不同，百戏书写可分为**历史书写、文学书写、宗教书写**三种主要类型。

### （一）历史书写

**历史书写**是指史书、政书、类书等历史文献中对百戏的记载与叙述。其主要特征是：以客观记录为基本取向，注重信息的准确性和系统性。

**1. 史书中的百戏书写**

史书是百戏历史书写的核心文献类型，包括纪传体正史和编年体史书。其中：

- **纪传体正史**：如《史记》《汉书》《后汉书》《三国志》《魏书》《隋书》等，在"本纪""志""传"等不同体例中均有百戏记载。
- **编年体史书**：如《资治通鉴》等，按年代顺序记载百戏相关事件。

史书中的百戏书写具有以下特点：

| 体例                     | 内容侧重           | 典型文献                         |
| ------------------------ | ------------------ | -------------------------------- |
| **本纪**                 | 帝王观赏百戏的活动 | 《汉书·武帝纪》                  |
| **志（礼乐志、音乐志）** | 百戏的制度、类型   | 《后汉书·礼仪志》《隋书·音乐志》 |
| **传**                   | 百戏艺人、相关人物 | 各史"佞幸传""倡优传"等           |

**2. 政书中的百戏书写**

政书是记载典章制度的文献，对百戏管理机构、演出规制有详细载录。主要包括：

- **《通典》**：杜佑撰，其中"乐典"部分详载百戏源流与类型；
- **《唐六典》**：载录唐代教坊制度及百戏管理；
- **《汉官典职》**（佚书，部分内容见于《后汉书》注引）：记载汉代宫廷百戏演出。

**3. 类书中的百戏书写**

类书是古代的百科全书式文献，汇集了大量百戏相关材料。主要包括：

- **《艺文类聚》**（唐·欧阳询等编）
- **《初学记》**（唐·徐坚等编）
- **《太平御览》**（宋·李昉等编）

类书的价值在于保存了大量已佚文献中的百戏材料。

### （二）文学书写

**文学书写**是指诗、赋、小说等文学作品中对百戏的描绘与表现。其主要特征是：以审美表达为基本取向，注重形象的生动性和情感的感染力。

**1. 赋中的百戏书写**

汉赋是百戏文学书写的重要载体，尤以京都大赋最为突出。代表作品包括：

| 作品           | 作者 | 百戏描写内容           |
| -------------- | ---- | ---------------------- |
| **《西京赋》** | 张衡 | 杂技、幻术、假面兽舞等 |
| **《东京赋》** | 张衡 | 宫廷宴乐中的百戏表演   |
| **《西都赋》** | 班固 | 角抵之戏               |
| **《两都赋》** | 班固 | 综合性百戏场面         |

张衡《西京赋》是汉代百戏文学书写的典范之作，对杂技、幻术、假面兽舞等各类百戏均有生动描绘【107】。

**2. 诗中的百戏书写**

唐诗中有大量涉及百戏的作品，主要包括：

- **描写宫廷百戏演出的诗作**：如描写教坊歌舞、宫廷宴乐；
- **描写民间百戏活动的诗作**：如描写节庆期间的杂耍、幻术表演；
- **以百戏为比喻或寄托的诗作**：借百戏表达人生感慨。

**3. 笔记小说中的百戏书写**

唐代笔记小说中保存了大量百戏材料，如《酉阳杂俎》《朝野佥载》《隋唐嘉话》等，记载了百戏艺人的轶事、百戏表演的奇闻等。

### （三）宗教书写

**宗教书写**是指宗教典籍、祭祀文献、礼仪文本中对百戏及相关内容的记载。其主要特征是：以宗教仪式或信仰实践为背景，百戏与宗教活动存在交叉或融合。

**1. 礼仪文献中的百戏书写**

《周礼》《仪礼》《礼记》等礼学经典及其注疏中，包含了与百戏相关的内容，特别是关于"散乐""傩仪"的记载。如《周礼·旄人》郑玄注云："散乐，野人为乐之善者，若今黄门倡矣，自有舞。"【108】

《后汉书·礼仪志》详细记载了东汉大傩仪的具体仪程【109】，其中的方相舞、十二兽舞等均属百戏范畴。

**2. 道教文献中的百戏书写**

道教斋醮仪式中常有歌舞、杂技表演，道教文献中也保存了相关记载。此外，道教对"幻术"有独特的理解和运用。

**3. 佛教文献中的百戏书写**

佛教传入中国后，寺院法会中常有百戏演出。敦煌文献中保存了大量唐代寺院百戏活动的记录。此外，佛教变文、讲唱文学与百戏也存在密切联系。

### （四）三种书写类型的比较

| 类型         | 书写主体   | 主要载体           | 核心特征 | 研究价值                     |
| ------------ | ---------- | ------------------ | -------- | ---------------------------- |
| **历史书写** | 史官、学者 | 史书、政书、类书   | 客观记录 | 了解百戏的制度、类型、源流   |
| **文学书写** | 文人、诗人 | 赋、诗、笔记小说   | 审美表达 | 感受百戏的艺术魅力、社会影响 |
| **宗教书写** | 祭司、僧道 | 礼仪文献、宗教典籍 | 仪式功能 | 理解百戏与宗教的互动关系     |

这三种书写类型各有侧重，相互补充，共同构成了汉唐百戏书写的完整图景。本文的研究将综合运用这三类文献，力求全面、深入地考察汉唐百戏的历史面貌与文化意涵。
